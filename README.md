# holy-grail
A short CSS code to help ya on holy-grail layout building.

## Basic-usage

```html
<div class="holy-grail holy-align-middle">

    <div class="holy-body">

        <div class="holy-row">
            <!-- your code goes here -->
        </div>
        
    </div>
        
</div>
```
